## Escenario elegido
Escenario B - Análisis de ventas.

## Descripción
En este trabajo se realizó un análisis simple de un dataset de ventas utilizando Python.

El objetivo fue practicar:
- uso de Git y GitHub
- trabajo colaborativo
- manejo de Jira
- uso de Google Colab

El script desarrollado permite:
- calcular ventas totales
- mostrar la venta más alta
- agrupar ventas por mes
- generar un gráfico

## Dataset
Se utilizó el archivo MOCK_DATA.csv con datos simulados de ventas.

## Estructura del proyecto

- datos/
- scripts/
- resultados/

## Ejecución

Para ejecutar el script:

python scripts/analisis_ventas.py

## Resultados
El programa genera un gráfico de ventas por mes dentro de la carpeta resultados.
